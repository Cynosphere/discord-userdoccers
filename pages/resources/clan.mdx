# Clan Resource

Clans are an extension of guilds that allow users to apply to join a clan's guild and use a clan's tag globally.

### Clan Object

###### Clan Structure

| Field                 | Type             | Description                                                                 |
| --------------------- | ---------------- | --------------------------------------------------------------------------- |
| id                    | snowflake        | The ID of the guild                                                         |
| name                  | string           | The name of the guild                                                       |
| tag                   | string           | The tag of the clan (2-4 letters)                                           |
| icon_hash             | ?string          | The guild's [icon hash](/reference#cdn-formatting)                          |
| playstyle             | integer          | Number representing a [playstyle](#playstyle)                               |
| member_count          | integer          | Approximate count of total members in the guild                             |
| description           | string           | Description of the clan, this can be different than the guild's description |
| search_terms          | array[string]    | A list of 3 or more interests/topics/traits                                 |
| game_ids              | array[snowflake] | Array of application IDs                                                    |
| badge                 | integer          | Number representing a [badge kind](#badge-kind)                             |
| badge_color_primary   | string           | The primary color of the badge as a hexadecimal color string                |
| badge_color_secondary | string           | The secondary color of the badge as a hexadecimal color string              |
| banner                | integer          | Number representing a [banner style](#banner-style)                         |
| brand_color_primary   | string           | The primary color of the banner and some text as a hexadecimal color string |
| brand_color_secondary | string           | The secondary color of the banner as a hexadecimal color string             |
| wildcard_descriptors  | array[string]    | A list of 3 words to describe the clan                                      |
| badge_hash            | string           | The clan's [badge hash](/reference#cdn-formatting)                          |
| banner_hash           | string           | The clan's [banner hash](/reference#cdn-formatting)                         |

## Endpoints

<RouteHeader method="POST" url="/clan/{guild.id}">
  Convert guild to clan
</RouteHeader>

Enables clan features for a guild

###### JSON Params

| Field                 | Type                                    | Description                                                                 |
| --------------------- | --------------------------------------- | --------------------------------------------------------------------------- |
| tag                   | string                                  | The tag of the clan (max 4 characters)                                      |
| game_application_ids  | array[snowflake]                        | Array of application IDs                                                    |
| search_terms          | array[string]                           | A list of interests from a predefined list of terms                         |
| play_style            | integer                                 | Number representing a [playstyle](#playstyle)                               |
| description           | string                                  | Description of the clan                                                     |
| wildcard_descriptors  | array[string]                           | A list of 3 words to describe the clan                                      |
| verification_form     | [verification form](#verification-form) | The verification form data shown to applicants                              |
| badge                 | integer                                 | Number representing a [badge kind](#badge-kind)                             |
| badge_color_primary   | string                                  | The primary color of the badge as a hexadecimal color string                |
| badge_color_secondary | string                                  | The secondary color of the badge as a hexadecimal color string              |
| banner                | integer                                 | Number representing a [banner style](#banner-style)                         |
| brand_color_primary   | string                                  | The primary color of the banner and some text as a hexadecimal color string |
| brand_color_secondary | string                                  | The secondary color of the banner as a hexadecimal color string             |
| badge_image           | [image data](/reference#cdn-data)       | The clan badge's image finalized as a PNG                                   |
