# Clan Resource

Clans are an extension of guilds that allow users to apply to join a clan's guild and use a clan's tag globally.

### Clan Object

###### Clan Structure

| Field                 | Type             | Description                                                                 |
| --------------------- | ---------------- | --------------------------------------------------------------------------- |
| id                    | snowflake        | The ID of the guild                                                         |
| name                  | string           | The name of the guild                                                       |
| tag                   | string           | The tag of the clan (2-4 characters)                                        |
| icon_hash             | ?string          | The guild's [icon hash](/reference#cdn-formatting)                          |
| playstyle             | integer          | The [play style](#play-style) of the clan                                   |
| member_count          | integer          | Approximate count of total members in the guild                             |
| description           | string           | Description of the clan; this can be different than the guild's description |
| search_terms          | array[string]    | 3 or more interests, topics, or traits for the clan                         |
| game_ids              | array[snowflake] | The IDs of the applications representing the games the clan plays           |
| badge                 | integer          | The [badge shown on the clan's tag](#badge-type)                            |
| badge_hash            | string           | The clan's [badge hash](/reference#cdn-formatting)                          |
| badge_color_primary   | string           | The primary color of the badge as a hexadecimal color string                |
| badge_color_secondary | string           | The secondary color of the badge as a hexadecimal color string              |
| banner                | integer          | The [banner representing the clan](#banner-style)                           |
| banner_hash           | string           | The clan's [banner hash](/reference#cdn-formatting)                         |
| brand_color_primary   | string           | The primary color of the banner and some text as a hexadecimal color string |
| brand_color_secondary | string           | The secondary color of the banner as a hexadecimal color string             |
| wildcard_descriptors  | array[string]    | 3 words to describe the clan                                                |

## Endpoints

<RouteHeader method="GET" url="//discovery/games/{game_type}">
  Get Clans
</RouteHeader>

Returns a list of [clan](#clan-object) objects for a [game type](#game-type).

###### Game Type

| Value    | Description              |
| -------- | ------------------------ |
| all      | All clans                |
| genshin  | Clans for Genshin Impact |
| valorant | Clans for Valorant       |

<RouteHeader method="GET" url="/discovery/{guild.id}/clan">
  Get Clan
</RouteHeader>

Returns a [clan](#clan-object) object associated with a guild if it has one.

<RouteHeader method="POST" url="/clan/{guild.id}">
  Create Clan
</RouteHeader>

Enables clan features for a guild. Returns a 204 empty response on success. Fires a [Guild Update](/topics/gateway-events#guild-update) Gateway event.

###### JSON Params

| Field                 | Type                                                                        | Description                                                                 |
| --------------------- | --------------------------------------------------------------------------- | --------------------------------------------------------------------------- |
| tag                   | string                                                                      | The tag of the clan (2-4 characters)                                        |
| game_application_ids  | array[snowflake]                                                            | The IDs of the applications representing the games the clan plays           |
| search_terms          | array[string]                                                               | 3 or more interests, topics, or traits for the clan                         |
| play_style            | integer                                                                     | The [play style](#play-style) of the clan                                   |
| description           | string                                                                      | Description of the clan; this can be different than the guild's description |
| wildcard_descriptors  | array[string]                                                               | 3 words to describe the clan                                                |
| badge                 | integer                                                                     | The [badge shown on the clan's tag](#badge-type)                            |
| badge_image           | [image data](/reference#cdn-data)                                           | The clan's badge finalized as a PNG                                         |
| badge_color_primary   | string                                                                      | The primary color of the badge as a hexadecimal color string                |
| badge_color_secondary | string                                                                      | The secondary color of the badge as a hexadecimal color string              |
| banner                | integer                                                                     | The [banner representing the clan](#banner-style)                           |
| brand_color_primary   | string                                                                      | The primary color of the banner and some text as a hexadecimal color string |
| brand_color_secondary | string                                                                      | The secondary color of the banner as a hexadecimal color string             |
| verification_form     | [membership screening](/resources/guild#membership-screening-object) object | The membership screening shown to applicants                                |

<RouteHeader method="GET" url="/clan/{guild.id}/settings">
  Get Clan Settings
</RouteHeader>

Returns a guild's clan settings.

###### Response Body

| Field                 | Type                                                                        | Description                                                                 |
| --------------------- | --------------------------------------------------------------------------- | --------------------------------------------------------------------------- |
| tag                   | string                                                                      | The tag of the clan (2-4 characters)                                        |
| game_application_ids  | array[snowflake]                                                            | Array of application IDs                                                    |
| search_terms          | array[string]                                                               | A list of interests from a predefined list of terms                         |
| play_style            | integer                                                                     | Number representing a [playstyle](#playstyle)                               |
| description           | string                                                                      | Description of the clan                                                     |
| wildcard_descriptors  | array[string]                                                               | A list of 3 words to describe the clan                                      |
| verification_form     | [membership screening](/resources/guild#membership-screening-object) object | The membership screening data shown to applicants                           |
| badge                 | integer                                                                     | Number representing a [badge kind](#badge-kind)                             |
| badge_color_primary   | string                                                                      | The primary color of the badge as a hexadecimal color string                |
| badge_color_secondary | string                                                                      | The secondary color of the badge as a hexadecimal color string              |
| banner                | integer                                                                     | Number representing a [banner style](#banner-style)                         |
| brand_color_primary   | string                                                                      | The primary color of the banner and some text as a hexadecimal color string |
| brand_color_secondary | string                                                                      | The secondary color of the banner as a hexadecimal color string             |

<RouteHeader method="POST" url="/clan/{guild.id}/disable">
  Disable Clan
</RouteHeader>

Disables a guild's clan features. Returns a 204 empty response on success. Fires a [Guild Update](/topics/gateway-events#guild-update) Gateway event.
